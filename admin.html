Google Maps

Clé API pour la carte intégrée sur la page Contact

Vérification…

Aperçu de la carte avec votre clé :

La clé est stockée dans Supabase et chargée automatiquement sur la page Contact. Activez Maps Embed API dans Google Cloud Console.

Comment obtenir une clé Google Maps gratuite ?
  1. Allez sur console.cloud.google.com
  2. Créez un projet (ou sélectionnez un existant)
  3. Menu → APIs & Services → Bibliothèque
  4. Recherchez "Maps Embed API" → Activer
  5. Menu → APIs & Services → Identifiants
  6. Cliquez "+ Créer des identifiants" → Clé API
  7. Copiez la clé et collez-la ci-dessus
💡 Google offre 28 000 chargements/mois gratuits — largement suffisant pour un site salon.

Sans clé API : La carte OpenStreetMap gratuite reste affichée avec un bouton "Ouvrir dans Google Maps". La clé Google Maps est optionnelle mais donne une meilleure expérience.
